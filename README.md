# Alumni Registration Form
## Technoledgy Used:
- Architecture (**MVC**)
- Nodejs ( **BackEnd** )
- Express ( **Web application framework** )
- ejs ( **Template engine** )
- DataBase ( **MySql / PostgreSQL**)

## For Deploy
- Set DB config in .env file
- Set NODE_ENV in .env file (optional)
- Set Port in .env
- Make uploads folder on root
- RUN ``npm start``
